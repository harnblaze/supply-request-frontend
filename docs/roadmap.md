# Roadmap (Frontend)

Цель: собрать **React SPA** для отслеживания заявок на закупку материалов с интеграцией по REST API (контракт из `docs/swagger.json`), таблицами на TanStack Table, формами на `react-hook-form` + `zod`, загрузкой файлов через `input type="file"` и скачиванием со статического `/uploads`.
